#### Quick start

* install vagrant

* install virtualbox

* install chef-sdk

* `echo 'eval "$(chef shell-init bash)"' >> ~/.bash_profile`

* `. ~/.bash_profile`

* `git clone https://github.com/vadikgo/chef-dev.git`

* `cd chef-dev`

* `gem install berkshelf`

* `rake`

* `vagrant up --provision`
